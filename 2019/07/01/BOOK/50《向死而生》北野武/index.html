《向死而生》北野武

首页 文学

设想倒是有一大堆，不过都是纸上谈兵，实际一去行动，什么也做不了。

# 4-84

总有一些人喜欢被操纵着，同时还有另一部分人喜欢操纵着别人。

我的生活就是工作，三百六十五日日如此，做的事情非常单调。

每个人都像是被附体一样，每一天都理所当然像机器似的生活着，可在猛然离开日常生活后我才发现，这竟然意外的使人觉得很幸福。

我想到那里走走，即使到那里走，没有任何意义。

动物，就是靠自己的力量行走，并吃下食物生存下去。不行的时候就被吃掉。

我的腿像戴上沉重的脚铐似的，同时又被困在床上，简直是双重的枷锁。

结果，就只是看着不如我的人，衡量位置关系罢了。

# 4-85

为了活下去，就要吃饭，可是现在所有地方都扭曲了。饥饿而死的人有很多。

自己的脑子里事后也考虑了很多事，突然就那样坐上了传送带，什么也没有改变。

只是活着这件事本身有价值，其他的没有。大概就是脑袋还在转，心脏还在跳，只是这些还有意义，其他的部分都是附加的。

人肯定是想着不如自己的人来确认自己的位置才放心，自己也沾染了这种劣根性。

到目前为止的人生，自己到底做了什么？

我觉得这绝对不是我，也不是为了她自己，仅仅是想让我高兴罢了。

好感度这种东西为什么有意义呢？这一点都不令人高兴。

不管是坏的意义还是好的意义，如果不留下任何痕迹，就没有任何意义。

继续活下去，如果处于被动为止的话，总有一天会继续不下去的。

无意识的度过时间是非常重要的，如果没有意识到这点，就不会消磨掉时间。

他们所做的绝大多数表面的工作，大抵都会是一种赚钱的手段，每一步都是有安排的，不做那个不行。因为社会就是这样扭曲被建立的，那种程度的安排不做不行。尽管内心觉得很傻。

我不觉得社会理念有什么不好，它也不过是一种宗教。那只是不会处理的人的失败。用脑袋思考而身体是朝着完全不同的方向行动的人不在少数。

我想要逃到某个地方轻松一下，不过，那之后要是不回来的话就有意思了。

我有我的生活方式，不到嘭的撞墙的时候，就不会改变方向。

发生事故，稍稍停下来，虽然一段时间速度会降下来，但恢复到平时时，就会又按原来的速度跑起来。那就是生活。如果被迫用不同的速度走不同的方向，那就已经不是那个人的人生了。是另外别的什么人的人生。

所谓的精神，根据自己的位置在哪儿，判断就会全然不同。

无论是战争还是生病，所谓的战场，归根结底都不的不是一个人的战斗。听取周围人的意见，即使死了，责任也只能由本人承担。

可以说，大家一起过桥的话就不会害怕桥塌了，现在已经变得在哪里是一样的了。

只要努力就会获胜，这句话失去价值的瞬间，任何差距都没办法加以区别了。

# 4-86

到处都是没有什么缘由而成立的大学，扔个石头就能砸到一个大学教授。

学校的教师曾经是令人刮目的职业，现在成了那些未通过企业的考试者不得已做的职业。

上层次的人没必要和平民一样去看大减价吧。

低级的家伙就会在低级的地方演出，本来有价值的东西也会廉价拍卖。

现在的人们已经连笑的能力都失去了。把矛盾看成理所当然的事情不奇怪的吗？

喝酒的一瞬间，就会说做什么都不要紧，是我的细胞在这么想的吧？

即使是作为思想或者哲学，也像个的到了一大堆玩具的孩子一样，从这个挑到那个眼花缭乱的，最后对哪个都不能好好地玩。而只得到一个玩具的孩子，就一定会爱惜的玩着。

各色各样的食物一个一个涌现出来就像连续更换电视频道那样，只有单纯的信号在做着反应。

这样的事情一个也不做的话，人类就到了陷入无底沼泽的时代了。

我们又不去赌博，却硬要把这些禁忌放在自己心中。

这和考驾驶执照时的性格测试是一样的。要是说真心话，就会打个叉。大家都知道说些原则上的话，所以最开始就决定还是画个圈。

仅仅是认为这样回答好所以这样回答。和实际是不同的，人们只是把本来按照步骤做的事情原原本本地作为问题，就成了没有实质的东西了。

即使是总理大臣也不能把每个问题都答出来。

把这种不起任何作用的事情作为工作还会给他们奖金，真实没办法。

大家都在随波逐流，他们没时间也没必要去考虑那些问题。

无望的社会也是同样做着无望的事情。

聚集这些数字，也只是徒劳，什么作用也不起。

只拘泥于形式，这样能够满足的事情就过多了。

官员们的工作就是做一些没有意思的事。搬到新的办公楼，也只是增加人数做着没有意义的事。

# 4-87

很多机关做着奇怪的事，还不起任何作用，只是在浪费税金而已。

叫来了文化人或者是大学教授等各类人听取意见，只能和舆论调查一样说出一些不得罪人的意见。

他们模模糊糊的有的理想，但从不考虑隐藏在背后的东西。他们很擅常盲目并相信并且吸收。

我们不能停止，所以会想大家还是一起来承担这样沉重的负担吧，要不是这样的话，就变成了不能做这种事的世界了。

一个人生活在社会中，只能是依靠别人或被别人依靠。

对于很多人来说，和老婆孩子一起愉快的生活就是他们活着的目的了。

现在的男人既不是丈夫，也不是父亲。仅仅是身边的好人罢了，大体上，自己工作转赚的钱全部被别人拿走，却很不在意。

就会像受到社会的制裁一样，没有办法，只能为了孩子做出牺牲。

结果，男人都变成了奴隶，作为奴隶，所以很轻松。什么也不用考虑，只要做就行了。现在的男人已经不能任性了，也不希望让人说那个家伙只考虑自己。

没有无限制的自由。自由不是白白的就能占为己有的东西。正因为如此，是价值最高的了。怎样自由的生存，就要怎样的限制自己。

我想真是出丑，也不是那么低贱的事情，归根结底不只是为了赚钱吗？

不知缘由的彷徨着，不知是好是坏，没有根本答案。

拼命工作赚钱，过着他们应该过的生活，这就是他们的梦想。

在这片土地上，如果你是一个平凡的人，从小屋子里脱身的可能性就是零，发挥出很大才能的人和没有能力的人的水准并没有那么大的差距。

多亏了宪法，让我们变得自由平等了，实际上变得更加没有自由，人与人之间的差距更大的社会。

让我们感觉到这个国家就好像一座大的监狱，在这个监狱中虽然说是人人平等，其中有特殊优待的囚犯头儿，即使是在最低劣的条件之中，逐渐的也一定会出现不出力而获得好处的家伙。

# 4-88

偶尔也会有主张正确言论的人，但是马上就会被排挤出去。他们被称作是顽固不通融的人或者不懂诙谐的人。

设想倒是有一大堆，不过都是纸上谈兵，实际一去行动，什么也做不了。

店长被不停的替换，但是小屋里边要做的事情没有改变。

暗地里这样做的悲惨，只是在表面上做的很好来敷衍过去。

在哪儿都是接受同样的教育，从哪儿毕业也都是发给同样的资格证。

人类也是一样的。虽说是大学毕业，但是也有像有 “虫眼” 蔬菜的家伙，即使做了很多改变，但是有虫眼就是有虫眼。大家都这样伪装着，也有人让他们就职。

人类要是这么温柔的动物的话，早就该灭绝了。能够存活到现在，是因为人类曾那么凶残的杀死了敌人。

真正在做自己喜欢做的事的人是不会有的。所谓的兴趣，是花费了一生的时间持续下去的才叫兴趣。

说这你这样的不可以去，不可以谈话，不可以主张的家伙，就像是有所有的权利那样指挥着别人。即使是总理大臣，不能担任的家伙也在担任着。

如果有命地话，那就尽情活着吧。

哪怕歪着脸也要咬紧牙关拼命地继续他那有趣的工作，无论这项工作是什么事情。
