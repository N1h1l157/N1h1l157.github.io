# 八维类别

Ti：为啥 - 科学方法 - 更注重 “我” 的逻辑

Te：有啥用 - 定位和规划 - 收集外部信息判断

Fi：我想要啥 - 最深的爱和信念 - “我” 的情感体验

Fe：他们要啥 - 群体和谐 - 普世大众的价值观

Si：记得是这样 - 记忆中的经验 - 对过去的联想

Se：长这样 - 现实、享乐 - 对当下的体验

Ni：也可以这样 - 规律 - 透过现象看本质

Ne：如果这样呢 - 可能性 - 对可能性进行猜测

# 四种方向

# N

Ni 与 Ne 密不可分，同样是直觉的使用，不同点仅在于维度。

Ni：想象，抽象；历时，预期，展望；创新；使命感，理想。

Ne：想象，抽象；共时，连锁联想，意识流；跳跃。

# S

Si 与 Se 的关系可以理解为 “Ni 与 Ne 的关系” 在现实层面的仿同。

Si：现实，实感；历时，经验，规则；传统。

Se：现实，实感；共时，经验，事实，感官。

# F

Fi 与 Fe 同样为情感偏好，不同点在于对信息加工处理后如何输出为决定和意见。

Fi：以自身情感为标杆，自我，独立。

Fe：以满足他者情感为目的，奉献。

# T

Ti 与 Te 为逻辑偏好，指向客观事理一方

Ti：内在逻辑，个人标准。

Te：外在公理，普世规律。

# 八大功能

# 第一功能

认知方式。最常使用，最擅长功能。但隐藏着内在缺陷和负面影响。

# 第二功能

处理方式。辅助第一功能，使内外互相联系，得以运作。

# 第三功能

支撑功能，疗伤功能，深层动机。

# 第四功能

弥补第一功能的先天缺陷面。

# 第五功能

通常测量值比第五位高。虚高。工具中的工具，打酱油的。

好像能用，但用着不舒服。

# 第六功能

深层判断依据，智慧洞察。

出问题时容易被激发。

# 第七功能

“骗子”。以为对自己很重要，其实不过可以拿来娱乐或骗别人。

内心抗拒使用的功能。

# 第八功能

“魔鬼”。与主功能联合作用将形成致命盲点，导致严重错误。

自然情况下用了就容易出问题的功能。

# 理解八维

# Ni 与 Ne

  1. Ni 内情直觉

  • Ni 是历时性的，是时间轴上的，并且方向向前，指向未来；是对未来的设想与不可知，与现时分离。

  • Ni 功能的使用表现为内在的关于未来的想象。最突出的例子即创作者在创作之前对作品大致形式的构想。

  • 音乐家事先在脑中合成一首完整的交响乐，确定曲风和节奏甚至细枝末节的和弦运作，画家对手头未完成化作的设想，作家对小说的预期和故事发展走向的设定等等，都是 Ni 的运作。

  1. Ne 外倾直觉

  • Ne 是共时性的，是对此时此刻眼前场景的信息收集，却凌驾于现象之上，与现实分离。

  • Ne 与 Se 将当前世界切成两部分，一部分由实体和现象（Se）构成，另一部分则由抽象与想象（Ne）构成。

# Si 与 Se

  1. Si 内倾感觉

  • Si 是历时性的，方向朝向过去，是对过去现象的采集与积累

  • 体现为个人经验、社会传统、规范准则等，是现实的、经验性的、接地气的和完全隶属过去的，与凌空对于未来的构想 Ni 形成鲜明对比。

  1. Se 外倾感觉

  • Se 与 Ne 同属当下，位于同一维度的不同位置。

  • Se 是对当前场景内切实可感到的一切信息的收集，是耳闻目见的事实的采集，与 Ne 的联想和想象形成对比。

# Fi 与 Fe

  1. Fi 内倾情感

  • Fi 以满足内在情感为动力，Fe 以满足外在情感为动力。

  • 将情感比作一股流水，Fi 的情感流流向自我，流向内在，成为被输入方；

  1. Fe 外倾情感
  • Fe 则相反，情感流流向他人，流向外在，成为输出方。

# Ti 与 Te

  1. Ti 内倾思考

  • Ti 以自身内在逻辑为标准

  • 可视作一个人内在的世界观与小宇宙的法则，完全个人化的逻辑建构

  1. Te 外倾思考

  • Te 以外在客观逻辑为标准

  • 更贴近众所周知的因果律、形式逻辑、公理。
