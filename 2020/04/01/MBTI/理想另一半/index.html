# SJ

# ESTJ

希望对方能幽默有趣，如果能搞笑一些再好不过了。

有担当，做出的承诺一定会履行。大多时候都会听我的，愿意在我面前服软认输。

# ISTJ

规划人生，一起进步，在一些细节里体现出爱意。

但可能不会用直接或者强烈的语言肢体表达。

# ESFJ

爱情比较传统，渴望踏实平稳的爱情生活，希望双方都是有规划有共同目标的。

刚开始会短暂的浪漫和激情，接下来就会很快进入一种平淡的节奏。

伴侣的感觉就像是温柔的臂弯，是最大的依靠吧，专注于每天的一点点小事，就很觉得很踏实很幸福了。

# ISFJ

喜欢自信阳光，比较理性的人，两个人可以一起成长，在各自擅长的领域发光发热，互相欣赏互相尊重。

可以一起玩，但也给彼此留有一定的独处时间。

会比较向往以结婚组建家庭为导向的，门当户对父母都认可的那种爱情。

# SP

# ESFP

真爱至上。会在爱情中表达自己当下炙热的感受，会热情表达对对方的喜爱之情，会体贴细致地为对方着想。

希望感情中会出现 “我在闹 ta 在笑” 这样的感觉。

# ISFP

喜欢动物和大自然，希望 ta 是温柔且细心的人，能陪我一起安静地躺在草坪上看星星。

在意我的感受，愿意去我的世界了解我，呵护我。

# ESTP

能够接受柏拉图式恋爱，但是自己没办法做到，在我这里爱和性是分不开的。

两个人谈恋爱其实也是一种博弈，要一起往前走一起进步，否则再深厚的感情都会因为一方的停滞不前被消磨殆尽。

# ISTP

希望 ta 独立强大，有自己的梦想，而且能够包容我的性格和缺点。

必要时一定要给彼此适当的个人空间。

# NT

# ENTJ

绝对的尊重，适度的建议，无声的陪伴，经常的惊喜，十分的浪漫，偶尔的示爱。

# INTJ

我只喜欢比我强的人。

# ENTP

不关乎其他，灵魂上的共鸣胜过一切。

# INTP

精神层次大于物质大于肉体，对方需要有独立思考的能力和渊博的知识，需要可以带领我帮助我的那种。

在外界可以表现的孤僻些，但是内心强大不 emo。

最好有相同的音乐品味以及审美，文笔好的话很加分。物质基础需要有享受生活的机会。

# NF

# ENFJ

希望 ta 性格有趣，开朗优秀，有闪光点，是会让我发自内心敬佩的人。

能适当的放养自己，给我足够的自由空间，依赖我，包容我。

# INFJ

性格温和，三观相近、善于沟通和表达自我感受很重要，我希望我能看懂并理解 ta 的基础上，还能喜欢 ta。

相比于喜欢优点我更关注彼此对缺点的包容程度。爱好可以截然不同，但是可以求同存异，会愿意走进我的世界了解我。

真诚和信任，是唯一能打动 “我” 的品质。

# ENFP

足够坦诚，足够真诚，互相鼓励，共同前行，愿意和我一起经历所有的曲折和坎坷，完全的坚定的支持我。

# INFP

想找灵魂伴侣，互相治愈成长。爱人大概等同于好朋友中的好朋友，先成为我的朋友才有可能成为我的恋人。

能吃到一起，聊到一起，玩到一起，在遇到困难的时候能够互相支持说‘我在’的人。
